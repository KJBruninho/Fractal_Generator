# Fractal Generator 🌀

Um gerador de fractais de alto desempenho desenvolvido em **C++**, focado na exploração visual de conjuntos matemáticos complexos.

## Sobre o Projeto
Este projeto permite a geração e visualização de fractais (como o conjunto de Mandelbrot). Utiliza C++ para garantir que os cálculos matemáticos iterativos sejam processados de forma eficiente, permitindo explorar a natureza recursiva destas estruturas.

## Tecnologias Utilizadas
* **Linguagem:** C++
* **Ambiente:** Desenvolvido com Visual Studio (inclui ficheiros `.sln`)
* **Conceitos:** Matemática Complexa, Recursividade, Renderização de Imagem.

## 📁 Estrutura do Repositório
* `/Fractal`: Contém o código-fonte principal (`.cpp`, `.h`).
* `Fractal.sln`: Ficheiro de solução do Visual Studio para carregamento rápido do projeto.

## Como Executar
1.  Clona o repositório:
    ```bash
    git clone [https://github.com/KJBruninho/Fractal_Generator.git](https://github.com/KJBruninho/Fractal_Generator.git)
    ```
2.  Abre o ficheiro `Fractal.sln` no **Visual Studio**.
3.  Compila o projeto em modo `Release` para obter a melhor performance de geração.
4.  Executa o binário gerado.
